# WHY torch with --index-url for CPU?
#   Default torch from PyPI includes CUDA binaries (~2.5GB).
#   The CPU-only version is ~200MB. We don't have a GPU, so we force CPU.
#   In requirements.txt we note this — the actual install flag goes in Dockerfile.
#
# Note: When installing locally, use:
#   pip install torch --index-url https://download.pytorch.org/whl/cpu
